# IAT School of Technology — Website

Static website for IAT School of Technology, Nairobi, Kenya.

**Live site:** https://daydroidmuchiri.github.io/iat-school-of-technology/

## Structure

- `index.html` — single-page site: utility bar, hero slider, quick-access tiles,
  animated stats, about, schools, programmes, admissions, news & events
  (with announcements and date-badged events), testimonials, why IAT, contact,
  multi-column footer
- `css/style.css` — all styles, responsive, no external dependencies
- `js/main.js` — hero slider, count-up stats, mobile nav, scroll-reveal, scroll-to-top

## Editing

The contact details (address, phone, email) in the Contact section of `index.html`
are placeholders — replace them with the school's real details.

Deployed with GitHub Pages from the `main` branch root.
